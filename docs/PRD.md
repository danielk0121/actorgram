# PRD — actors

## 공통

- 앱 화면의 모든 정보는 한글로 표기한다.
- 최상단 메뉴바를 가진다. 앱 이름과 파비콘은 좌측, 메뉴는 우측에 정렬한다.
- 좌측 로고(앱 이름 또는 파비콘)를 클릭하면 홈 화면(배우 화면)으로 이동한다.
- 배우 메뉴: 배우를 검색하여 출연 영화 목록을 확인하는 화면이다.
- 영화 메뉴: 영화를 검색하여 영화 목록을 확인하는 화면이다.
- 홈 화면은 배우 화면과 동일하며, 상단바에 별도 홈 메뉴를 두지 않는다.
- 사진 검색 메뉴: 사진을 업로드하여 배우 또는 영화를 검색하는 화면이다.

## 사진 검색 도메인

- 사용자는 사진을 업로드하여 배우 또는 영화를 검색할 수 있다.
- 사진 업로드 시 샘플 검색 결과를 표시한다. (톰 크루즈 배우 검색 결과 기준)
- 샘플 화면은 사진 업로드 후 사용자가 보게 될 결과 화면의 예시이다.

## 배우 도메인

- 사용자는 배우 이름 또는 배역 이름으로 배우를 검색할 수 있다.
- 배우 정보로 이름, 출생연도, 국적, 데뷔일, 프로필 대표 사진을 표시한다.
- 배우 정보 하단에 해당 배우의 영화 속 이미지가 1줄 가로 스크롤로 표시된다.
- 영화 속 이미지는 최대 10개까지 표시되며, 더보기 버튼이 있다.

## 영화 도메인

- 영화는 주연배우 목록을 가진다.
- 검색 결과는 배우 정보 섹션 + 영화 목록 섹션으로 구성된다.
- 배우 화면의 영화 목록은 그리드 형태로 표시된다. 모바일(420px 이하)에서는 1줄에 2개씩 표시된다.
- 영화 화면의 영화 목록은 1줄에 1개씩 표시되며, 포스터와 정보가 가로로 나란히 배치된다.
- 영화 화면은 영화 제목, 장르, 감독으로 검색할 수 있다.
- 영화 정보로 제목, 연도, 장르, 감독, 작가, 관람등급, 상영시간, 개봉일, 제작국가를 표시한다.
- 배우 화면의 영화 카드에는 영화 포스터 1장(1:1 비율)과 검색 기준 배우의 배역 이미지 3장을 표시한다.
- 배우 화면: 영화 카드에 검색 기준이 된 배우의 배역 정보만 표시한다.
- 영화 화면: 영화 카드에 해당 영화의 주연배우 전체를 표시한다. 각 주연배우에는 프로필 이미지를 표시한다.
- 배우 화면: 영화 카드 클릭 시 영화 화면으로 이동하여 해당 영화를 검색한 결과를 표시한다.
- 영화 화면: 주연배우 클릭 시 배우 화면으로 이동하여 해당 배우를 검색한 결과를 표시한다.
- 배우 도메인의 각 영화 배역 정보에는 영화 속 배역 이미지 3장이 포함된다.
- 앱 홈화면 초기 상태는 "톰 크루즈" 검색 결과를 예시로 표시한다.
